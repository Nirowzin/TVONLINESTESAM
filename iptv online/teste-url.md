## Como Testar sua URL:

1. Faça login como admin (senha: admin123)
2. Cole sua URL no campo "URL da Lista M3U"
3. Clique em "Carregar URL"
4. Aguarde o carregamento

## Problemas Comuns:

- **URL não responde**: Verifique se a URL está correta e acessível
- **Erro de CORS**: O sistema agora usa proxy interno para contornar isso
- **Conteúdo vazio**: Verifique se a URL retorna um arquivo M3U válido
- **Timeout**: Alguns servidores podem demorar para responder
